# Cyber Task HUD

A high-performance, stateless task management interface featuring a 3D-responsive, sci-fi inspired aesthetic.

This application is a visual demonstration of integrating modern UI/UX principles with a robust Django backend. It utilizes CSS 3D transforms, JavaScript-based parallax, and a custom neon-themed design system to provide an immersive user experience.

## Technical Architecture

* **Backend:** Django (Python 3.12+)
* **Frontend:** Vanilla HTML5, CSS3, and JavaScript
* **Visuals:** Custom 3D Parallax engine with animated grid systems
* **Icons:** Phosphor Icons (Vector-based)
* **Database:** SQLite3 (Integrated ORM)

## Design Philosophy

The interface is built around a "Heads Up Display" (HUD) concept, utilizing glassmorphism and motion graphics to create depth. The layout is optimized for high-impact visual presentations in a portfolio context.
